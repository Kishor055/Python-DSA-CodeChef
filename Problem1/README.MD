🔍 Search an Element in an Array
Problem Description

You are given an array A of size N and an element X.
Your task is to determine whether the array contains the element X.

Input Format

First line: Two integers N and X

Second line: N space-separated integers (array A)

Output Format

Print:

"YES" if element X is present

"NO" otherwise

Constraints

1 ≤ N, X ≤ 10⁵

1 ≤ Aᵢ ≤ 10⁵

Sample Input 1
5 3
7 3 5 2 1

Sample Output 1
YES

Sample Input 2
5 10
7 3 5 2 1

Sample Output 2
NO
